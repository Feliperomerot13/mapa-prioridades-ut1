# Mapa de Prioridades · UT1

Aplicación estática para construir el Mapa de Prioridades personal: OKR, actividades, impacto, esfuerzo, MoSCoW y plan de reenfoque a 2 semanas.

## Despliegue en GitHub Pages

1. Sube estos archivos a un repositorio de GitHub.
2. Ve a **Settings → Pages → Build and deployment → Source**.
3. Selecciona **GitHub Actions**.
4. Haz commit/push a la rama `main`.
5. Revisa la pestaña **Actions** hasta que el despliegue termine.

La URL final suele quedar así:

```txt
https://TU-USUARIO.github.io/NOMBRE-DEL-REPOSITORIO/
```

## Uso en la sesión

- La información queda guardada localmente en el navegador del participante.
- En celular, la matriz final se puede deslizar horizontalmente.
- Para evidencia rápida en móvil, usar **Modo captura / pantallazo**.
- Para PDF, usar **Imprimir / guardar PDF** y seleccionar orientación horizontal cuando el sistema lo permita.
